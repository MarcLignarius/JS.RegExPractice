RegEx Practice

Password Validation

Enter a password in the field below:

  • Password must be 6 to 12 characters in length
  • Password must have at least one upper case letter (A-Z)
  • Password must have at least one lower case letter (a-z)
  • Password must have at least one digit (1-9)

URL Validation

Enter a URL in the field below.

  • URL must start with http, https or ftp followed by ://
  • URL must match a valid domain name
  • URL could contain a port specification (http://www.sitepoint.com:80)
  • Could contain digit, letter, dots, hyphens, forward slashes, multiple times

HTML Tag Validation

Enter a HTML tag in the field below.

  • The start tag must begin with < followed by one or more characters and end with >
  • The end tag must start with < / (no space) followed by one or more characters and end with >
  • We must match the content inside a TAG element
